See It Live

The goal of this project was to create the in-vehicle User Experience for the new generation of GMC and Chevy Trucks. I was responsible for creating the user flow, navigation, functionality, graphics, motion behaviour (animations, micro interactions), and real-time implementation using Unity/Kanzi Studio. The project ran for 18 months from concept to production and included relocation to Detroit, Michigan.
