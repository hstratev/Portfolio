about.
See It Live

The project started with an internal company request to create the UX for a platform showing the properties of the battery set in electric vehicles. From iconography and basic battery function display, the project evolved into a complex system displaying battery status, overall charge and faults, module statistics, connectivity status, and BMC sleep/wake cycles. Total project duration was three months from start to final CES showcase.
